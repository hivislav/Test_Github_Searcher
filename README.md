# Test_Github_Searcher
App for search repositories on Github.
Functionality:
- find public repos on github
- open link in browser possibility
- download zip repo possibility
- open downloaded zip

Stack:
- Kotlin
- Clean Architecture
- MVVM
- Coroutines
- Dagger2
- Room
- Retrofit2
- Github REST Api
- Material
- Animated splash screen (for api 31+)
- Coil
- View Binding
